### 概要
**#コンパス 戦闘摂理解析システム**の2025年12月23日現在での
- 実装キャラ
- 実装カード
- 実装ステージ

をまとめたものです．  
それぞれ下記エンドポイントを提供しています．

|エンドポイント|取得データ|
|-|-|
|`api/compass/cards`|実装カード|
|`api/compass/heroes`|実装キャラ|
|`api/compass/stages`|実装ステージ|

本 API は静的ホスティング（GitHub Pages）での運用を前提として設計されています．  
GitHub Pages 上にデプロイしている関係上，クエリによる検索機能は提供しておらず，各エンドポイントからは該当データをすべて一括で取得する形式になっています．  
**※ 本APIで取得できる名称・画像等の権利は，各権利者に帰属します．**
